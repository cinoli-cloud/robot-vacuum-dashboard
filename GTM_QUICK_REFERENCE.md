### 1. 价格对比表格（主功能）

**如何使用**:
1. 打开看板
2. 查看55个产品的实时价格
3. 点击品牌按钮筛选特定品牌
4. **点击任意价格** → 跳转到购买页面验证

**核心指标**:
- 官网价格（MSRP基准）
- Amazon价格（主要竞品渠道）
- Walmart价格（美国大众市场）
- Costco价格（会员制渠道）

- 24h变化（价格波动趋势）

- ⚪ 灰色 = 无变化

## 🚨 价格警报设置

### 启用警报系统

```bash
cd python
python price_alerts.py
```

### 默认警报规则

1. **大幅降价**: 下降≥5%
   - 提示: 竞品促销启动
   - 行动: 考虑跟进

2. **大幅涨价**: 上涨≥5%
   - 提示: 供应紧张/需求旺盛
   - 行动: 可能的提价机会

3. **超低价格**: <$500
   - 提示: 性价比机会
   - 行动: 评估竞争力

### 自定义警报

编辑 `python/alert_config.json`:
```json
{
  "rules": [
    {
      "name": "Eufy Competitor Price Drop",
      "type": "price_decrease",
      "threshold": -3.0,
      "brands": ["Roborock", "Dreame"],
      "enabled": true
    }
  ]
}
```
